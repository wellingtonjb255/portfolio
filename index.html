Portfólio

Abaixo estão alguns projetos que desenvolvi durante o curso.

Projeto de banco de dados

Este relatório apresenta as soluções para a atividade prática da disciplina de Banco de Dados. A primeira etapa aborda a criação do Modelo Entidade-Relacionamento (MER) para um sistema de rede de hotéis, incluindo a definição de entidades, atributos, relacionamentos e cardinalidades, conforme as regras de negócio especificadas. A segunda etapa foca na implementação de um banco de dados para uma locadora de veículos, contendo os scripts SQL para a criação das tabelas e a execução de cinco consultas específicas para extrair informações relevantes do sistema, como manutenções, faturamento total, veículos mais locados e pagamentos pendentes.

Dashboard de Custos Operacionais em Excel

Este projeto consiste em um dashboard operacional, desenvolvido em Microsoft Excel, para o monitoramento e análise de custos de serviços de roçagem. O painel interativo exibe indicadores-chave de desempenho (KPIs), como a quantidade de roçagens realizadas e o valor total gasto. Através de filtros dinâmicos, é possível visualizar as despesas detalhadas por categoria (frete, hospedagem, combustível, etc.) para cada localidade e período, oferecendo uma ferramenta visual e eficaz para a gestão de custos e tomada de decisões.
